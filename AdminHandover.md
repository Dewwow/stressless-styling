# Stressless Styling - Admin Handover Guide

## Overview
Stressless Styling is a managed package for hair salons to manage appointments, services, and customer preferences. It includes an Experience Cloud site for customer and employee self-service.

## Package Information
- **Namespace:** dewwow
- **API Version:** 60.0

## Experience Site
- **Site Name:** stressless
- **Template:** Build Your Own (LWR)
- **Purpose:** Customer and employee portal for appointment booking and management

## Custom Objects
- **Preferences__c** - Stores customer hair and service preferences

## Key Features (Planned)
- Appointment scheduling
- Service catalog management
- Customer preferences tracking
- Google authentication for login

## Development Notes
- This project uses CumulusCI for development and deployment
- Scratch orgs are used for development (`cci flow run dev_org --org dev`)
